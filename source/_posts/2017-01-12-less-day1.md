## less overview

### variables

* Variables

```less
@nice-blue: #5B83AD;
@light-blue: @nice-blue + #111;

#header {
  color: @light-blue;
}
```
output 

```CSS
#header {
  color: #6c94be;
}
```

* mixin

```less
.bordered {
  border-top: dotted 1px black;
  border-bottom: solid 2px black;
}

#menu a {
  color: #111;
  .bordered;
}

.post a {
  color: red;
  .bordered;
}

```
* Nested Rules (tlnr)

* Nested Directives and Bubbling

Directives such as media or keyframe can be nested in the same way as selectors.   
Directive is placed on top and relative order against other elements inside the same ruleset remains unchanged. This is called bubbling.

```less
.screen-color {
  @media screen {
    color: green;
    @media (min-width: 768px) {
      color: red;
    }
  }
  @media tv {
    color: black;
  }
}
```

```css
@media screen {
  .screen-color {
    color: green;
  }
}
@media screen and (min-width: 768px) {
  .screen-color {
    color: red;
  }
}
@media tv {
  .screen-color {
    color: black;
  }
}
```
