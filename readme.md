# This hexo repo handbook-ish

### **Step1:** Clone project
```
$ git clone -b master https://github.com/haoaa/hexo.git .
```

### **Step2:** npm install
```
$ npm install
```

### **Step3:** write some md file under the `source/_posts`
```
$ hexo new something-about-sass
``` 
### **Step4:** backup

- commit source

 ```
 $ npm run git -- "your commit comments goes here"
 ```
- deploy commond

 ```
 $ npm run reload 
 ```
